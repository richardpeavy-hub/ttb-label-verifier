SETUP
- Download and install Tesseract-OCR and AutoIt.  Both installers are included on this repository.  Use default settings for the installations.

- Download and open Label Checker - AutoIt.zip and then drag the included folder anywhere you prefer.

- Open the folder and double-click Sandbox.au3.  It will run and scan all the images within the Labels folder.  Text output currently gets placed in the ExtractedText folder.

- This program is incomplete.  Details are in the Label Checker Documentation file.
